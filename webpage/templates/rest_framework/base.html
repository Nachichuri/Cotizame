¡Bienvenidx! Esta es la parte divertida donde se puede jugar con todos los datos disponibles en Divisa.me

Esto es una API RESTful súper sencilla pero que nos permite interactuar con los datos. Si bien tiene muchas funcionalidades, de manera pública están disponibles los métodos GET, HEAD y OPTIONS. Podés consultar las cotizaciones haciendo el llamado directamente al subdirectorio de cada banco, o podes bajar un JSON para tener local todos los datos hasta el momento de la descarga (recordá que también podés descargarlo en otros formatos desde la tabla de cada banco en la web principal).

Tip: si querés una API MCQLR (Más Cruda Que La Realidad) podés encontrarla haciendo click acá.

A su vez, podés consultar aún más específicamente si proporcionás una fecha exacta. Por ejemplo 'https://divisa.me/api/$(banco)/2020-04-28' te va a devolver un JSON con la cotización del 28 de abril de 2020 de cada divisa en ese banco. Te invito a que pruebes todas las opciones y combinaciones disponibles, y que uses esta data como más te sirva.
